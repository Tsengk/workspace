# Python3 基础练习

> 对应输出物模板：1.1
> 知识点：变量、分支、循环、函数、列表、字典

## 练习目标

基于 GSDS / ICAOS_TOOL 相关脚本场景，写出基础练习代码并运行通过。

## 运行方式

```bash
python3 <文件名>.py
```

## 文件说明

| 文件名 | 内容 |
|--------|------|
| `变量与类型.py` | int/float/str/bool/list/tuple/dict、类型转换、type()、f-string |
| `条件分支.py` | if/elif/else、逻辑运算符、三元表达式、按相机类型分发策略 |
| `循环控制.py` | for/while、break/continue、zip/enumerate、嵌套循环、列表推导式 |
| `函数定义.py` | 参数默认值、*args/**kwargs、多值返回、lambda、类型注解、LEGB 作用域 |
| `列表操作.py` | CRUD、切片/索引/步长、sort/sorted、map/filter/reduce、浅/深拷贝 |
| `字典操作.py` | CRUD、get/setdefault/pop、keys/values/items、推导式、defaultdict/Counter |

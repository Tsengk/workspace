# 玫瑰病害分类模型 — 部署手册

## 项目概述

基于 YOLOv8n-cls 的玫瑰叶片病害图像分类模型，支持 8 种病害类别识别。

## 环境要求

| 项 | 最低配置 | 推荐配置 |
|---|---------|---------|
| Python | 3.8+ | 3.10+ |
| 内存 | 8 GB | 16 GB |
| 磁盘 | 5 GB | 10 GB |
| GPU | 无（CPU 可跑） | NVIDIA 4GB+ 显存 |

## 目录结构

```
yolo_project/
├── train.py              # 训练脚本
├── predict.py            # 预测脚本
├── datasets/             # 数据集
│   ├── train/            #   训练集（8 个子目录 = 8 个类别）
│   │   ├── black_spot/
│   │   ├── downy_mildew/
│   │   ├── fresh_leaf/
│   │   ├── insect_hole/
│   │   ├── insects_infected/
│   │   ├── powdery_mildew/
│   │   ├── rose_mosaic/
│   │   └── rose_rust/
│   └── val/              #   验证集（同上结构）
├── runs/classify/        # 训练输出
│   └── rose_disease/
│       └── weights/
│           └── best.pt   # 最佳模型
└── docs/                 # 文档
```

## 部署步骤

### 1. 克隆项目

```bash
git clone <仓库地址>
cd yolo_project
```

### 2. 准备数据集

确保 `datasets/` 目录结构正确：

```
datasets/
├── train/
│   ├── 类别A/   (图片直接放里面)
│   └── 类别B/
└── val/
    ├── 类别A/
    └── 类别B/
```

新增类别只需在 `train/` 和 `val/` 下各建一个文件夹，放入图片即可。

### 3. 创建虚拟环境

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 4. 安装依赖

```bash
pip install ultralytics matplotlib
```

### 5. 训练

```bash
python train.py
```

关键参数（可在 `train.py` 中修改）：

| 参数 | 含义 | 建议值 |
|-----|------|-------|
| model | 预训练权重 | yolov8n-cls.pt（快）/ yolov8s-cls.pt（准）|
| data | 数据集路径 | datasets/ |
| epochs | 训练轮数 | 50-100 |
| imgsz | 图片尺寸 | 224（快）/ 640（准）|
| batch | 批次大小 | CPU: 8, GPU: 32 |
| device | 设备 | cpu / 0（GPU编号）|

训练完成后模型保存在 `runs/classify/rose_disease/weights/best.pt`。

### 6. 验证准确率

```bash
# 详细报告（整体 + 各类别 + 混淆矩阵）
python predict.py --val

# YOLO 内置验证
yolo classify val model=runs/classify/rose_disease/weights/best.pt data=datasets/
```

### 7. 预测

```bash
# 单张图片
python predict.py /path/to/leaf.jpg

# 整个文件夹
python predict.py /path/to/folder/

# YOLO CLI 方式
yolo classify predict model=runs/classify/rose_disease/weights/best.pt source=图片路径
```

### 8. 常见问题

**Q: Shared Memory 报错**
修改 `train.py` 中 `workers=0`

**Q: 显存不足**
减小 `batch=4` 或 `batch=2`

**Q: 训练太慢**
先用 `imgsz=224` 快速验证，确认无误后再用 `imgsz=640` 正式训练

**Q: 新增类别**
在 `datasets/train/` 和 `datasets/val/` 下各建新文件夹放入图片，重新训练即可

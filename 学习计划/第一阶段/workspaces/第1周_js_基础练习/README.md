# JavaScript 基础练习

> 对应输出物模板：1.2
> 知识点：变量、函数、对象、数组、模块导入导出

## 练习目标

基于 MRCT / 测试任务软件 / ICAOS_TOOL 前端场景，写出基础练习代码并运行通过。

## 运行方式

```bash
node <文件名>.js
```

## 文件说明

| 文件名 | 内容 |
|--------|------|
| `变量声明.js` | var/let/const、块级作用域、变量提升、基本类型 typeof、模板字符串 |
| `函数与箭头函数.js` | 函数声明/表达式、箭头函数 this、默认参数、剩余参数、回调、闭包 |
| `对象操作.js` | 字面量/属性访问、CRUD、解构、展开、Object.assign/深拷贝、可选链 ?. |
| `数组方法.js` | push/splice/slice、forEach/map/filter/reduce/find/sort、flatMap、解构 |
| `模块导入导出.js` | CommonJS (require/exports) + ESM 动态 import()，拆分 cameraUtils 模块 |
